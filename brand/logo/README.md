# Logo Files

Drop your logo variants here:

- `logo-primary.svg` — Full logo with text (for title slides, headers)
- `logo-mark.svg` — Just the symbol/icon (for favicons, small spaces)
- `logo-wordmark.svg` — Just the text (for minimal contexts)

SVG format is preferred — it scales to any size and content skills can embed it directly into presentations, infographics, and video compositions.

If you only have PNG/JPG, drop those in and skills will use them. SVG just gives better results.
