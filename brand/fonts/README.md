# Font Files

Self-hosted font files go here. Supported formats: `.woff2`, `.woff`, `.ttf`, `.otf`

Recommended structure:
- `display.woff2` — Display/heading font
- `body.woff2` — Body text font
- `mono.woff2` — Monospace font (for code)

If you use Google Fonts or Adobe Fonts instead of self-hosting, just note the font names in `brand/VISUAL.md` under the Typography section. Skills will generate the correct `@import` or `@font-face` declarations.
