# AI Slop Detection Patterns

Comprehensive list of AI-generated content patterns. Organized by category with severity levels and suggested fixes.

---

## Severity Levels

- **Critical**: Immediately recognizable as AI-generated. Must be fixed before publishing. Each instance deducts 5-10 points from the anti-slop score.
- **Warning**: Common in AI content but also appears in human writing. Fix when possible. Each instance deducts 2-4 points.
- **Info**: Subtle pattern that may or may not be a problem depending on context. Review but do not auto-penalize. Each instance deducts 0-1 points.

---

## Category 1: Opening Cliches

**Severity: Critical**

| Pattern | Fix |
|---|---|
| "In today's fast-paced world..." | Cut entirely. Start with your actual point. |
| "In today's rapidly evolving [landscape/world/industry]..." | Cut entirely. Start with a specific observation or claim. |
| "In an era where..." | Cut entirely. |
| "Let's dive in..." | Cut. Just start. |
| "Let's explore..." | Cut. Just start. |
| "Have you ever wondered..." | Replace with a direct statement or specific question. |
| "Imagine a world where..." | Replace with a concrete example or scenario. |
| "Picture this:" | Replace with a direct description without the meta-framing. |
| "It's no secret that..." | If it is not a secret, just state the fact. |
| "We all know that..." | If we all know it, why are you telling us? Make your new point. |
| "As we all know..." | Same as above. Cut. |
| "In this article, we will explore..." | Cut. The reader knows they are reading an article. Start with the content. |
| "Welcome to our comprehensive guide to..." | Cut. Get to the value. |
| "Are you struggling with..." | Replace with a specific problem statement. |
| "Whether you're a beginner or an expert..." | Pick your audience and speak to them. |
| "If you're like most [people/developers/marketers]..." | Be specific about who you are talking to. |
| "[Topic] has become increasingly important..." | Why? Show, don't tell. Lead with the evidence. |
| "The world of [topic] is constantly changing..." | What changed, specifically? Lead with the specific change. |
| "In the ever-changing landscape of..." | Cut. Name the specific change you are writing about. |
| "[Number] ways to [do something]" without a hook | Add a reason why the reader should care. |
| "Ready to [transform/supercharge/level up] your..." | Cut the hype. State the specific benefit. |

## Category 2: Filler Phrases

**Severity: Critical** (first 3), **Warning** (rest)

| Pattern | Fix |
|---|---|
| "It's worth noting that..." | Cut the phrase. State the thing directly. |
| "It goes without saying that..." | If it goes without saying, don't say it. Cut. |
| "Needless to say..." | Same as above. Cut entirely. |
| "It's important to note that..." | Cut the phrase. Just state the note. |
| "It's crucial to understand that..." | Cut. State the understanding directly. |
| "It should be mentioned that..." | Mention it without the preamble. |
| "It's essential to remember that..." | Cut. State the thing. |
| "At the end of the day..." | Cut. State your conclusion directly. |
| "When it comes to [topic]..." | Cut. Start with what you want to say about the topic. |
| "In terms of..." | Usually replaceable with a more direct construction. |
| "The fact of the matter is..." | Cut. State the fact. |
| "When all is said and done..." | Cut. State the conclusion. |
| "As a matter of fact..." | Cut. State the fact. |
| "By and large..." | Cut or replace with a specific quantifier. |
| "For all intents and purposes..." | Cut. State what you mean directly. |
| "In a nutshell..." | Cut if used as filler. Acceptable in a genuine summary. |
| "To put it simply..." | Just put it simply without announcing it. |
| "With that being said..." | Cut. Transition more naturally. |
| "Having said that..." | Cut or replace with "But" or "However." |
| "That being said..." | Same as above. |
| "On the other hand..." | Acceptable sparingly. Flag if used more than twice. |
| "All things considered..." | Cut. State the consideration. |

## Category 3: Overused Transitions

**Severity: Warning** (flag if any appears more than once in the same piece)

| Pattern | Fix |
|---|---|
| "Furthermore..." | Replace with a specific connection between ideas. |
| "Moreover..." | Same. Or just start the next sentence. |
| "Additionally..." | Same. |
| "In addition to this..." | Cut. Start the next point. |
| "On top of that..." | Cut or replace with specific connection. |
| "Not only... but also..." | Acceptable once. Flag if repeated. |
| "First and foremost..." | Just say "First" or cut entirely. |
| "Last but not least..." | Just state the last point. |
| "Moving on..." | Cut. The reader moves on by reading the next sentence. |
| "Now, let's look at..." | Cut. Just present the next section. |
| "Without further ado..." | Cut. |
| "That said..." | Acceptable sparingly. Flag if repeated. |

## Category 4: Generic Metaphors and Analogies

**Severity: Warning**

| Pattern | Fix |
|---|---|
| "Think of it as a [simple analogy]..." | Replace with a specific, illuminating analogy or cut. |
| "It's like a [cliche analogy]..." | Make the analogy specific and surprising, or remove it. |
| "...is the backbone of..." | Replace with a specific description of the relationship. |
| "...is the cornerstone of..." | Same as above. |
| "...is a double-edged sword..." | Describe the specific tradeoffs instead. |
| "...at the forefront of..." | Name what they are actually doing. |
| "...paving the way for..." | Describe the specific mechanism. |
| "...pushing the boundaries of..." | What boundaries? Be specific. |
| "...a breath of fresh air..." | What makes it refreshing? Be specific. |
| "...the tip of the iceberg..." | Describe what else is below. |

## Category 5: Vocabulary Tells

**Severity: Warning**

Words and phrases that appear disproportionately in AI-generated content:

| Word/Phrase | Fix |
|---|---|
| "leverage" (as a verb) | Use "use," "apply," or a more specific verb. |
| "unlock" (as in "unlock potential") | Describe the specific benefit. |
| "game-changer" | Describe the specific impact with numbers or examples. |
| "paradigm shift" | Describe the specific change. |
| "cutting-edge" | What makes it advanced? Be specific. |
| "state-of-the-art" | Same. Name the specific capability. |
| "seamless" | Describe the actual experience. |
| "robust" | What makes it robust? Be specific. |
| "streamline" | Describe what it actually simplifies. |
| "empower" | Describe the specific capability it gives people. |
| "holistic" | Describe what it covers specifically. |
| "synergy" | Describe the specific benefit of the combination. |
| "innovative" | Show the innovation, don't label it. |
| "revolutionary" | Same. Show, don't tell. |
| "delve" | Use "explore," "examine," or "look at." |
| "tapestry" | Use a more concrete metaphor or describe directly. |
| "landscape" (metaphorical) | Name the specific domain or market. |
| "myriad" | Use "many" or give a specific number. |
| "plethora" | Use "many" or give a specific number. |
| "navigate" (metaphorical) | Describe the specific actions. |
| "foster" | Describe how, specifically. |
| "realm" | Use the specific domain name. |
| "multifaceted" | Describe the specific facets. |
| "nuanced" | Show the nuance, don't label it. |

## Category 6: Structural Tells

**Severity: Warning**

| Pattern | Detection | Fix |
|---|---|---|
| All paragraphs same length | Measure paragraph word counts; flag if >60% are within 15% of the mean | Vary paragraph lengths: mix 1-sentence paragraphs with 3-5 sentence paragraphs. |
| Predictable cadence | Every sentence follows subject-verb-object pattern | Vary sentence structures: questions, fragments, compound sentences, inversions. |
| Three-point pattern | Every section has exactly 3 sub-points | Vary the number of supporting points per section (2, 4, 5). |
| Listicle flatness | Every item in a list has the same structure and depth | Vary list item lengths: some brief, some expanded with examples. |
| Symmetrical sections | All sections are exactly the same length | Let important sections be longer. Less important sections can be shorter. |
| No short paragraphs | Minimum paragraph length is 3+ sentences | Add occasional 1-sentence paragraphs for emphasis and rhythm. |

## Category 7: Em-Dash Abuse

**Severity: Warning** (>2 per 500 words), **Info** (<2 per 500 words)

AI models tend to overuse em-dashes. Detection:

- Count em-dashes (---) in the content
- Calculate the rate per 500 words
- Flag if the rate exceeds 2 per 500 words

Fix: Replace some em-dashes with:
- Commas (for parenthetical asides)
- Periods (to break long sentences)
- Colons (when introducing an explanation)
- Parentheses (for truly tangential notes)

---

## Category 8: Passive Voice Overuse

**Severity: Info** (<15% of sentences), **Warning** (15-30%), **Critical** (>30%)

Detection:
- Identify sentences with passive constructions ("was done," "is used," "has been implemented")
- Calculate the percentage of passive sentences
- Flag specific sentences with suggested active rewrites

Fix: Convert to active voice.
- Before: "The system was designed to handle high traffic."
- After: "We designed the system to handle high traffic."
- Before: "The feature was implemented last quarter."
- After: "The team shipped the feature last quarter."

Exceptions: Passive voice is acceptable when:
- The actor is unknown or unimportant ("The data was corrupted")
- The emphasis should be on the object ("The patient was treated successfully")
- Scientific or formal writing conventions require it

## Category 9: Closing Cliches

**Severity: Critical**

| Pattern | Fix |
|---|---|
| "In conclusion..." | Cut. Just deliver the conclusion. |
| "To sum up..." | Cut. |
| "To wrap things up..." | Cut. |
| "In summary..." | Cut. If you need a summary, use a "TL;DR" section. |
| "As we've seen..." | Cut. The reader knows what they have seen. |
| "At the end of the day..." | Cut. State the takeaway directly. |
| "The bottom line is..." | Cut the phrase, keep the bottom line. |
| "So there you have it." | Cut entirely. |
| "What are you waiting for?" | Replace with a specific, non-hype CTA. |
| "Start your journey today!" | Replace with a concrete next step. |
| "The future is bright for..." | Replace with a specific prediction or call to action. |
| "Only time will tell..." | State your actual prediction or recommendation. |

## Scoring Formula

See `scoring.md` for how these detections translate into the anti-slop score.

```
Base score: 100
- Each critical issue: -5 to -10 points
- Each warning issue: -2 to -4 points
- Each info issue: -0 to -1 points
- Minimum score: 0
```

The score reflects how many patterns were detected weighted by severity. A score of 100 means zero slop detected. A score below 50 indicates pervasive AI-generated patterns.
