# Format Transformation Matrix

This matrix shows how well each source format transforms into each target format. Use it to recommend transformations and set expectations about effort level.

---

## Rating Key

- **Natural** -- The transformation is straightforward. Content maps well from source to target with minimal restructuring.
- **Moderate** -- The transformation works but requires meaningful adaptation. Some content must be added, removed, or restructured.
- **Challenging** -- Significant rework needed. The transformation is possible but the target format requires substantially different content than what the source provides.
- **Poor** -- Not recommended. The formats are too different for the transformation to produce good results without essentially creating new content.

## Transformation Effort Estimates

Approximate effort for each difficulty level (assuming a 1500-word blog post as the baseline source):

| Difficulty | Time Estimate | What's Involved |
|---|---|---|
| Natural | 10-20 minutes | Extract, restructure, add platform-specific framing |
| Moderate | 20-45 minutes | Significant restructuring, add new context or framing, adapt substantially |
| Challenging | 45-90 minutes | Near-complete rewrite, add substantial new content, fundamentally different structure |
| Poor | 90+ minutes | You are essentially creating new content that happens to share a topic |

## Recommended Transformation Bundles

Common "1-to-many" packages that work well together:

### Blog Post Bundle
Starting from a blog post, create:
1. X thread (Natural) -- key insights as a tweetstorm
2. LinkedIn post (Natural) -- professional angle with personal take
3. Newsletter section (Natural) -- curated summary with commentary
4. Reddit post (Moderate) -- community-framed version of the key insight
5. Infographic (Moderate) -- visual summary of data or framework

**Total additional effort**: ~90-120 minutes for 5 derivative pieces

### Video Content Bundle
Starting from a video script or recording:
1. Blog post (Natural) -- written version with added detail
2. X thread (Natural) -- key moments as individual tweets
3. LinkedIn post (Moderate) -- lesson or takeaway with professional framing
4. Slide deck (Moderate) -- visual summary for presentations

**Total additional effort**: ~60-100 minutes for 4 derivative pieces

### Podcast Episode Bundle
Starting from a podcast transcript:
1. Blog post (Natural) -- structured written version of the conversation
2. X thread (Moderate) -- best quotes and insights
3. LinkedIn post (Moderate) -- key takeaway with guest attribution
4. Newsletter section (Moderate) -- episode summary and personal commentary
5. Audiogram clips (Natural) -- short audio snippets with captions for social

**Total additional effort**: ~90-120 minutes for 5 derivative pieces

### Presentation Bundle
Starting from a slide deck:
1. Blog post (Natural) -- expand speaker notes into full narrative
2. Video script (Natural) -- adapt presentation flow for recorded format
3. X thread (Moderate) -- key slides as individual tweets with commentary
4. Infographic (Natural) -- condense the visual story

**Total additional effort**: ~60-100 minutes for 4 derivative pieces

## When NOT to Repurpose

Some content should not be repurposed:

- **Time-sensitive reactions**: A hot take about today's news does not become a blog post next week
- **Platform-specific content**: A Reddit AMA or a LinkedIn poll does not translate to other formats
- **Low-performing source material**: If the original did not resonate, repurposing it amplifies the miss
- **Deeply personal content**: A vulnerable LinkedIn post may feel inauthentic as a polished blog
- **One-dimensional content**: If the source only has one point, there is not enough substance for multiple formats
